# Ejercicio: Condicionales en Java

## Objetivo
Practicar estructuras condicionales simples y anidadas en Java.

## Instrucciones
1. Crea un programa llamado `Condicionales.java`.
2. Pide al usuario su edad y muestra si puede votar (mayor o igual a 18).
3. Declara dos números y muestra cuál es mayor, o si son iguales.
4. Verifica si un número es positivo, negativo o cero.
